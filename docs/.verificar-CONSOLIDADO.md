# ✅ Workflow: Verificação

## Quando Usar
Validar saúde do projeto antes de commit/push.

## Comandos por Contexto

### Durante Desenvolvimento (Auto-executável)
```bash
npm run check        # Lint + types (< 1s)
```

### Antes de Commit (Auto-executável)
```bash
npm run test:quick   # Testes falhos + imports (~10s)
```

### Antes de Push dev/main (SOLICITAR ao usuário)
```bash
npm run verify       # TODOS os testes + lint + types (~4min)
```

**⚠️ IMPORTANTE**: IA **NUNCA** executa `verify` sem permissão explícita.

npm run security:secrets  # Detecta vazamento de credenciais
