# Roadmap Estratégico

Este documento descreve o planejamento de features futuras e a direção de longo prazo do produto.

> **Nota**: Prioridades sujeitas a alteração conforme feedback da comunidade.

---

## Backlog Futuro (Candidate Releases)

### Features de Performance (Gamificação)
**Status**: [Em Análise]
Sistema de recompensas visuais para incentivar o progresso.
- Sistema de XP por tarefa concluída.

- Persistência de conquistas.

### Grade Manager (Gestão de Notas)
**Status**: [Planejado]
Monitoramento de desempenho acadêmico.
- Scraping de boletim.
- Projeção de médias necessárias.
- Alertas de risco de reprovação.

---

## Processo de Priorização

Uma feature entra no roadmap quando satisfaz:
1.  **Valor Claro**: Resolve um problema real do aluno.
2.  **Viabilidade Técnica**: Compatível com as restrições de arquitetura (Local-First).
3.  **Manutenibilidade**: Não introduz complexidade excessiva de suporte.
