# Privacy Policy - Central Univesp Extension

**Effective Date:** January 6, 2026  
**Version:** 2.10.0

## Overview

Central Univesp is a browser extension designed exclusively for UNIVESP (Universidade Virtual do Estado de São Paulo) students. This Privacy Policy explains how the extension handles user data.

## Data Collection

**We do NOT collect, transmit, or share any personal data.**

### Data Storage (Local Only)

All data generated by the extension is stored **locally** on your device using Chrome's `chrome.storage.local` API:

- **Course Information:** Course names, week numbers, activity titles
- **User Preferences:** UI settings (density mode, auto-pin preferences)
- **Backup Files:** Exported JSON files created by the user

### Data Never Leaves Your Device

- ❌ No telemetry or analytics
- ❌ No remote servers
- ❌ No third-party services
- ❌ No tracking cookies

## Permissions Explained

The extension requests the following Chrome permissions:

| Permission | Purpose |
|-----------|---------|
| `storage` | Store user preferences and course data locally |
| `sidePanel` | Display the extension's side panel interface |
| `scripting` | Extract course content from AVA pages |
| `activeTab` | Navigate to specific activities within AVA |
| `downloads` | Export backup files (user-initiated) |

### Host Permissions

- `https://sei.univesp.br/*` - Auto-fill credentials on SEI portal
- `https://ava.univesp.br/*` - Extract course and activity data from AVA

## User Control

### Data Sovereignty

Users have full control over their data:

- **Export:** Download all data as JSON backup
- **Import:** Restore data from backup file
- **Factory Reset:** Permanently delete all local data

### How to Delete Your Data

1. Open extension settings
2. Navigate to "Gerenciar Dados" (Manage Data)
3. Click "Reset Total (Factory Reset)"
4. Confirm deletion

## Security

- **Trusted Types:** All dynamic HTML is sanitized to prevent XSS attacks
- **Content Security Policy:** Enforced strict CSP to block inline scripts
- **No Remote Code:** Extension does not load external scripts

## Changes to This Policy

We will notify users of privacy policy changes through extension updates. Continued use after updates constitutes acceptance.
