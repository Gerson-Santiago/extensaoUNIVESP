# Visão do Produto (Product Vision)

## Missão
Prover uma interface unificada e eficiente para gestão acadêmica, eliminando o atrito de navegação em sistemas legados e devolvendo tempo de estudo ao aluno.

---

## Princípios Norteadores (The Axioms)

### 1. Soberania de Dados (Local-First)
- **Privacidade Absoluta**: Dados sensíveis nunca deixam a máquina do usuário.
- **Independência**: O sistema funciona sem dependência de backends proprietários.

### 2. Ética na Automação
- **Facilitador, não Executor**: A ferramenta automatiza o *acesso* e *organização*, jamais o *consumo* de conteúdo educacional (ex: não assiste aulas pelo aluno).

### 3. Excelência Técnica
- **Qualidade Industrial**: Tratamos este projeto open-source com rigor de software enterprise (Testes, CI/CD, Arquitetura Limpa).

---

## Proposta de Valor
1.  **Centralização**: Dashboard único para Cursos, Prazos e Notas.
2.  **Contexto**: Navegação inteligente que entende a estrutura do AVA.
3.  **Performance**: Interface instantânea, sem refresh de página.
